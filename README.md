# Ai-Based-Disease-Pridiction-By-Symtoms
AI-based system that predicts possible diseases using 2–5 symptoms. It analyzes inputs with a machine learning model and shows likely conditions with confidence scores. Includes a user-friendly interface, risk indicator, and basic guidance to promote early awareness and timely medical action.

Disease Prediction by Symptoms

📌 Problem Statement
Many individuals ignore early symptoms due to lack of awareness, leading to late diagnosis.
Build a system that predicts potential diseases using minimal user input (2–5 symptoms).

💡 Solution
This project predicts possible diseases based on user symptoms using a machine learning model.
Develop Website with Prediction Dashboard
⚙️ Technologies Used
- Python
- Machine Learning
- HTML,CSS,JS
  
🚀 Features
- Input symptoms
- Predict disease
- Simple user interface
- Basic Recommendation
🧠 How it Works
User enters symptoms → Model processes → Disease is predicted

▶️ How to Run
1. Open index.html
2. Enter your Name,symtoms
3. Analyse the Disease

📊 Dataset
Kaggle :Disease Prediction Dataset
