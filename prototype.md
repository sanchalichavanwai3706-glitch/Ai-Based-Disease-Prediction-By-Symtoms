 Prototype Explanation

 📌 Overview
This prototype demonstrates a basic disease prediction system using symptoms.

 ⚙️ How it Works

1. User enters symptoms
2. System checks symptoms against dataset
3. Machine learning model predicts disease
4. Result is displayed to user

 💻 Features
- Simple input system
- Fast prediction
- Easy to use

 📷 Example

Input: fever, cough, headache  
Output: Flu

 ⚠️ Limitations
- Limited dataset
- Accuracy depends on data quality
- Not a replacement for medical advice

 🚀 Future Scope
- Improve accuracy
- Add more diseases
- Build full app (website/mobile)
