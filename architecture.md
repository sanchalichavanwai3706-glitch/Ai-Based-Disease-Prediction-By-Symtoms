# System Architecture

## 📌 Overview
This project predicts diseases based on user symptoms using a machine learning approach.

🔄 Workflow

User Input (Symptoms)
        ↓
Data Preprocessing
        ↓
Machine Learning Model
        ↓
Prediction Output (Disease)

 🧩 Components

 1. User Interface
- Takes symptoms as input from user

 2. Backend Processing
- Cleans and processes input data
  
 3. Machine Learning Model
- Matches symptoms with dataset
- Predicts most probable disease

 4. Output
- Displays predicted disease to user
  
 📊 Data Flow
User → Input Symptoms → Model → Disease Prediction
